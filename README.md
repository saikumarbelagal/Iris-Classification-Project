# Iris Flower Classification Using Machine Learning

## Project Overview
This project implements supervised machine learning classification using the Iris dataset.

## Dataset
- Iris Dataset
- 150 samples
- 4 features
- 3 classes (Setosa, Versicolor, Virginica)

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
- Cross Validation

## Results
- Logistic Regression Accuracy: 100%
- Random Forest Accuracy: 100%
- Average Cross Validation Score: 96.67%
- ROC-AUC Score: 1.0

## Tools Used
- Python
- Jupyter Notebook
- Scikit-Learn
- Pandas
- Matplotlib
- Seaborn
